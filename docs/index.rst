Welcome to pyjsgf's documentation!
==================================

Release v\ |release|

JSpeech Grammar Format (JSGF) compiler, matcher and parser package for Python.

JSGF is a format used to textually represent grammars for speech recognition engines. You can read the JSGF specification `here <https://www.w3.org/TR/jsgf>`_.
